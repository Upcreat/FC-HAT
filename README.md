# FC-HAT

The official code of Information science paper: FC-HAT: Hypergraph Attention Network for Functional Brain Network Classification

## Getting Started

### Requirements

- Python version >= 3.6
- PyTorch version >= 1.7.1
- DGL
- networkx
- numpy
- pandas
- scikit-learn
- scipy
- sklearn

## Usage

run 'train.py'
